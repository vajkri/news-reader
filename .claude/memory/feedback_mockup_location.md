---
name: mockup-location
description: HTML mockups go in .planning/assets/ with datetime-prefixed names (YYYY-MM-DD-HHMM-name.html)
type: feedback
---

Place HTML mockups and design assets in `.planning/assets/`, not in phase directories. Name files with a datetime prefix: `YYYY-MM-DD-HHMM-name.html` (e.g. `2026-03-29-1430-feed-toolbar.html`).

**Why:** User preference for centralized assets with chronological sorting.
**How to apply:** When creating mockups, always use `.planning/assets/` as the target directory and prefix the filename with the current date and time dash descriptive name.
