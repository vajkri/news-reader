Proposal C: Expandable Search Icon

  • Placement: A compact search icon in the toolbar right side. Click or press / to reveal a full-width search bar that slides down between toolbar and table.
  • Collapsed state: Toolbar stays compact and clean. Search icon is always visible as a small button.
  • Expanded state: Full-width bar with muted background slides down. Input auto-focuses. Esc closes and clears.
  • Active search indicator: When search is active but collapsed, the search icon turns blue with a dot indicator. Status bar shows the active term.
  • ADHD consideration: Fewer visible controls in default state reduces visual noise. But hidden search could be harder to discover and remember.
  • Trade-off: Cleanest default state, but search requires an extra click to reveal. The / keyboard shortcut mitigates this for power users. The animation adds a micro-interaction that could feel either polished or slow depending on preference.
