Purpose: These pure library modules are the foundation for fetching articles from sites without RSS feeds. Separated from schema/cron wiring so they can be developed and tested independently.

    **src/lib/fetch-article-meta.ts** (per D-01, D-02, D-03):
    - Add `import "server-only"` at top
    - Import `* as cheerio from "cheerio"` and `estimateReadTime` from `./readtime` and `ParsedArticle` type from `./rss`
    - Export async function `extractArticleMeta(url: string, lastmod: Date | null): Promise<ParsedArticle | null>`
    - Fetch the URL with `{ headers: { "User-Agent": "NewsReader/1.0 (news aggregator)" }, signal: AbortSignal.timeout(15000) }`
    - If `!res.ok` return null
    - Load HTML with `cheerio.load(html)`
    - Extract `title` from `meta[property="og:title"]` content attr, fallback to `<title>` text, fallback to `"Untitled"`
    - Extract `description` from `meta[property="og:description"]` content attr, fallback to `meta[name="description"]` content attr, fallback to null
    - Extract `thumbnail` from `meta[property="og:image"]` content attr, fallback to `meta[name="twitter:image"]` content attr, fallback to null
    - Extract `publishedAt`: first try JSON-LD `"datePublished"` via regex `/"datePublished"\s*:\s*"([^"]+)"/` on all `script[type="application/ld+json"]` text, then try `meta[property="article:published_time"]` content attr, then fall back to `lastmod` parameter. Validate parsed date with `!isNaN(parsed.getTime())`.
    - Return `{ guid: url, title, link: url, description, thumbnail, publishedAt, readTimeMin: estimateReadTime(description) }`
    - Wrap entire body in try/catch, return null on any error

    **src/lib/sitemap.ts** (per D-04, D-05, D-10):
    - Add `import "server-only"` at top
    - Import `XMLParser` from `fast-xml-parser`, `extractArticleMeta` from `./fetch-article-meta`, `ParsedArticle` type from `./rss`
    - Define `const MAX_AGE_MS = 7 * 24 * 60 * 60 * 1000`
    - Export async function `fetchSitemap(sitemapUrl: string, pathPattern: string): Promise<ParsedArticle[]>`
    - Fetch sitemapUrl with `{ headers: { "User-Agent": "NewsReader/1.0 (news aggregator)" }, next: { revalidate: 0 } }`
    - Parse XML with `new XMLParser({ ignoreAttributes: false })`
    - Normalize `urlset.url` to array: `const rawUrls = result?.urlset?.url; const urls = Array.isArray(rawUrls) ? rawUrls : rawUrls ? [rawUrls] : [];`
    - Filter by path pattern: `const pattern = pathPattern.replace(/\*/g, ""); candidates = urls.filter(u => u.loc?.includes(pattern))`
    - Split candidates into `withLastmod` (has `lastmod` string) and `withoutLastmod`
    - For `withLastmod`: filter to `new Date(u.lastmod).getTime() >= cutoff`, then `Promise.allSettled` calling `extractArticleMeta(u.loc, new Date(u.lastmod))`
    - For `withoutLastmod`: `Promise.allSettled` calling `extractArticleMeta(u.loc, null)`, then filter results where `publishedAt` is null OR `publishedAt.getTime() >= cutoff`
    - Collect all fulfilled non-null results into return array

    - Export async function `fetchArticles(source: FetchSource): Promise<ParsedArticle[]>`
    - Body: `const strategy = strategies[source.sourceType] ?? strategies.rss; return strategy(source);`
    - Export the `FetchSource` type
