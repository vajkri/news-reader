## What Was Built

**`triggerEnrichment` server action** (`src/lib/actions.ts`): Calls `/api/enrich` with `CRON_SECRET` header server-side. Returns `{ ok, enriched?, error? }`. Falls back to `http://localhost:3000` if `NEXT_PUBLIC_APP_URL` unset.

**`EnrichNowButton`** (`'use client'`): Outline Button variant. Uses `useTransition` + `useState` for non-blocking server action call. Shows "Enriching..." while pending, "Enrichment failed. Try again." for 3 seconds on error, then resets. Calls `router.refresh()` on success.

**`StatusBar`** (Server Component): Flex space-between layout with `--muted` background and `--border` border. Shows new article count (green-500 when > 0, foreground-secondary when 0) and last enriched time. Always renders `EnrichNowButton` on the right.

**`PendingSection`** (Server Component): Returns `null` when articles array is empty. Renders amber left border (3px, `border-amber-600`), `opacity-60` dimming, "Pending enrichment (N)" header, and per-article rows with title + "Awaiting enrichment" italic label.
