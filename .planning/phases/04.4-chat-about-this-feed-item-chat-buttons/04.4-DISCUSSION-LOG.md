## Pre-discussion: User-initiated feature addition

Before discussing the planned gray areas, the user described a new shortcut feature: when chat opens from a feed item, there should be a special prompt that fetches the full article content and creates an ADHD-friendly, detailed summary (1-2 min read). Motivation: avoid visiting distraction-heavy original sites with ads.

**User's choice:** Feature added to phase scope as "Read this for me" shortcut.
