## Decisions Made

- `curl -f` flag ensures each step fails immediately on HTTP errors, preventing silent failures

## User Setup Required

GitHub repository requires two manual configuration steps before the workflow can run:

1. **Secret:** Add `CRON_SECRET` as a repository secret (Settings > Secrets and variables > Actions > Secrets) with the same value as the Vercel environment variable
2. **Variable:** Add `APP_URL` as a repository variable (Settings > Secrets and variables > Actions > Variables) set to `https://news-reader-eta.vercel.app`

After setup, verify by triggering manually from Actions tab using "Run workflow".
