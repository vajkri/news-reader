**3. src/components/features/briefing/DateStepper.tsx** ('use client')

Must have `'use client'` directive at top.

Imports: `useRouter`, `useSearchParams` from `next/navigation`; `format`, `subDays`, `addDays`, `isToday`, `parseISO`, `isValid` from `date-fns`; `Button` from `@/components/ui/button`; `ChevronLeft`, `ChevronRight` from `lucide-react`.

Logic:
- Read date from `useSearchParams().get("date")`
- Parse with `parseISO`, validate with `isValid`, fallback to `new Date()`
- `goTo(date)`: if `isToday(date)`, `router.push("/briefing")`; else `router.push(\`/briefing?date=\${format(date, "yyyy-MM-dd")}\`)`
- Disable next button when `isToday(currentDate)` (per locked decision: no future dates)

JSX structure:
```tsx
<div className="flex items-center gap-2">
  <Button variant="outline" size="icon" onClick={() => goTo(subDays(currentDate, 1))} aria-label="Previous day">
    <ChevronLeft className="h-4 w-4" />
  </Button>
  <span className="text-sm font-semibold min-w-[140px] text-center text-[var(--foreground)]">
    {isToday(currentDate) ? "Today" : format(currentDate, "MMM d, yyyy")}
  </span>
  <Button variant="outline" size="icon" onClick={() => goTo(addDays(currentDate, 1))} disabled={isToday(currentDate)} aria-label="Next day">
    <ChevronRight className="h-4 w-4" />
  </Button>
  {!isToday(currentDate) && (
    <Button variant="ghost" size="sm" onClick={() => goTo(new Date())}>
      Today
    </Button>
  )}
</div>
```

Page function signature:
```typescript
export default async function BriefingPage({
  searchParams,
}: {
  searchParams: Promise<{ date?: string }>;
}) {
```

Date resolution:
```typescript
const params = await searchParams;
const dateParam = params.date;
const selectedDate = dateParam && isValid(parseISO(dateParam))
  ? parseISO(dateParam)
  : new Date();
const windowStart = startOfDay(selectedDate);
const windowEnd = endOfDay(selectedDate);
```

Prisma query:
```typescript
const articles = await prisma.article.findMany({
  where: {
    enrichedAt: { not: null },
    publishedAt: { not: null },
    importanceScore: { gte: 4 },
    publishedAt: {
      gte: windowStart,
      lte: windowEnd,
    },
  },
  orderBy: { importanceScore: 'desc' },
  take: 10,
  include: { source: { select: { name: true, category: true } } },
});
```

Note: the `publishedAt` where clause needs both `not: null` and the date range. In Prisma, combine these with AND:
```typescript
where: {
  enrichedAt: { not: null },
  importanceScore: { gte: 4 },
  AND: [
    { publishedAt: { not: null } },
    { publishedAt: { gte: windowStart, lte: windowEnd } },
  ],
},
```

Serialize and group:
```typescript
const serialized = JSON.parse(JSON.stringify(articles)) as ArticleRow[];
const topicGroups = groupArticlesByTopic(serialized);
```

JSX layout using section-container pattern:
```tsx
<div className="section-container py-6">

        <TopicGroup key={group.topic} group={group} />

</div>
```
